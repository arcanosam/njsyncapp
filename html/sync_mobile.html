Para conexão do seu dispositivo, utilize-se das orientações desta página
How to Enable USB Debugging Mode on Android que listamos abaixo resumidamente:

 
  1. Para Android 2.0-2.3.x
    1. Configurações
    2. Aplicações
    3. Desenvolvimento
    4. Marque Depuração USB
    5. Clique OK
  1. Para Android 3.0-4.1.x
    1. Configurações
    2. Opções de Desenvolvedor
    3. Marque Depuração USB
    4. Clique OK
  1. Para Android 4.2 e acima
    1. Configurações
    2. Sobre o telefone ou Sobre o tablet
    3. Clique na opção Número da versão
      Até aparecer a mensagem:Você é um desenvolvedor
    4. Volte a tela anterior
    5. Clique em Opções de Desenvolvedor
    6. Marque Depuração USB
    7. Clique OK
Em seguida, conecte o cabo USB no computador
